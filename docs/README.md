## It's really easy!

#### Via HTML
```html
<script src="https://unpkg.com/handsfree"></script>
```

#### Or NPM
```bash
npm i handsfree
```

#### Then in your project
```html
<script>
  const HandsfreeModule = require('handsfree')
  const handsfree = new HandsfreeModule({autoPointers: true, autoStart: true})
</script>
```
